11. What is a Promise?

a promise is a asynchronoues operation which can be in pending, fullfilled or rejected.
it allows handling of async operation more cleanly than callbacks

const myPromise = new Promise((resolve, reject) => {
  const success = true;

  if (success) {
    resolve("Operation was successful!"); // Sets state to fulfilled
  } else {
    reject("Something went wrong."); // Sets state to rejected
  }
});

myPromise
  .then((message) => {
    console.log("Success: " + message);
  })
  .catch((error) => {
    console.log("Error: " + error);
  })
  .finally(() => {
    console.log("Process completed.");
  });
