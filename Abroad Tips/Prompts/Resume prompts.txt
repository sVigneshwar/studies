# Resume Prompts

## Core prompt list — use one per request:

- Rewrite summary for [Job Title]: concise, keyword-focused, and recruiter-ready.
- Convert responsibilities into metric-driven achievements (use PAR/PAR format).
- Provide ATS keyword suggestions from a pasted job description and example placements.
- Reframe experience for a career transition, highlighting transferable impact.
- Full resume audit: call out weak language, redundancy, and ATS risks.
- Recommend a layout that prioritizes recent, relevant experience for fast scanning.
- Rewrite work history bullets to align with a specific job posting.
- Build an ATS-friendly Technical Skills section organized by category.
- Draft a short headline + subheadline stating role target and value proposition.
- Role-play as a hiring manager: list immediate changes to get an interview.

## Extras:

- Ask for before/after examples for direct copy-paste.
- Optionally request a confidence score for suggested edits.
